    variables

    susy

    $susy: () !default;
    View source

    Description

    User custom configuration for Susy

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    columns

    The number of columns in the grid

    		Integer4
    gutters

    The ratio of a gutter based on a column

    		Float0.25
    math

    Either fluid (percentiles) or static (any unit can be used for column-width)

    		String'fluid'
    column-width

    Only relevant when math option is set to static

    		Length or Nullnull
    spread

    The spread of an element across edge gutters: either narrow (none), wide (one), or wider (two)

    		String'narrow'
    context

    The spread of a container around edge gutters: either narrow (none), wide (one), or wider (two)

    		String'narrow'

    Example

    All available options (to their default value)

      $susy: (
    'columns': 12,
    'gutters': 1/4,
    'math': 'fluid',
    'column-width': null,
    'spread': 'narrow',
    'context': 'narrow',
  );

    Used by

    • [function] susy-settings

    susy-span (aliased as span )

    @function susy-span($span, $config: ()) { ... }
    View source

    Description

    Calculate the width of any grid-span. This is the primary function in Susy, and can be used anywhere you need to span columns on the grid: width for grid contents, or margin and padding to push, pull, and pad your elements.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $span

    Shorthand expression to define the width of the span. Contains a unitless number for column-span, 'of $n' for available grid columns [optional], 'at $n' for spanning asymmetrical grids [optional], and 'narrow', 'wide', or 'wider' for defining the spread over available gutters (none, one, or two)

    		List—none
    $config

    Optional map of Susy grid configuration settings

    		Map()

    Returns

    Length —

    Return either a fluid % value when math setting is fluid or an explicit length if math is static

    Example

    Span half the grid

      .foo
    width: susy-span(6 of 12);
  }

    Span a specific segment of asymmetrical grid

      .foo {
    width: susy-span(3 at 3 of (1 2 3 5 8));
  }

    Requires

    • [function] susy-settings
    • [function] su
    • [function] su-sum

    Used by

    • [function] span

    susy-nested (aliased as nested )

    @function susy-nested($span, $config: ()) { ... }
    View source

    Description

    Return a subset of asymmetrical columns to use for nesting context.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $span

    Shorthand expression to define the width of the subset span. Contains a unitless number for column-span, 'at $n', 'first', or 'last' for location on the grid, and optional 'of $n' for available grid columns.

    		List—none
    $config

    Optional map of Susy grid configuration settings

    		Map()

    Returns

    List —

    Return a subset list of asymmetrical columns.

    Example

    Span a specific segment of asymmetrical grid

      $context: susy-nested(3 at 3 of (1 2 3 5 8)); // returns `(3 5 8)`

    Throws

    • Asymmetrical grids always require a location.

    • The susy-nested function is not needed for symmetrical grids

    Requires

    • [function] susy-settings
    • [function] su-is-symmetrical
    • [function] su-slice

    Used by

    • [function] nested
